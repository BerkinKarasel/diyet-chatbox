# 🍏 Diyet Chatbox

Bu proje, **Streamlit + Hugging Face BloomZ** modeliyle çalışan basit bir diyet öneri chatbotudur.

## 🚀 Kullanım

1. Repo'yu klonla:
   ```bash
   git clone https://github.com/kullanici/diyet-chatbox.git
   cd diyet-chatbox
   ```

2. Bağımlılıkları yükle:
   ```bash
   pip install -r requirements.txt
   ```

3. Uygulamayı çalıştır:
   ```bash
   streamlit run app.py
   ```

4. Tarayıcıda `http://localhost:8501` adresinden erişebilirsin.

## ☁️ Streamlit Cloud'da Deploy

- [Streamlit Cloud](https://streamlit.io/cloud) hesabı aç.
- Bu repo'yu GitHub'a yükle.
- `New app` → repo'yu seç → birkaç dakika içinde çalışır.
